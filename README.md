# ToyStore
## An e-commerce store created using:
- React functional Components
- React hooks

## How to use: 
`npm install` then `npm start`

## Pages and Functionality:
- Product Listing page
- Product Detail Page
- Pop up modal
- Add item to cart
- Remove item from cart
- Clear cart
- CSS transitions
- Update item quantity in cart
- Pay using Paypal
